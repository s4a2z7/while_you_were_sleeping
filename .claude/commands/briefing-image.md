# 브리핑 이미지 생성

canvas-design Skill로 브리핑 카드를 생성합니다.

## 사용법

```
/briefing-image TSLA
/briefing-image NVDA
/briefing-image AAPL
```

## 파라미터

- **$ARGUMENTS**: 종목 코드 (예: TSLA, NVDA, AAPL)

## 출력 사양

- **크기**: 1200x630 (OG 이미지 표준)
- **테마**: 다크 모드 (네이비 + 보라색 그라디언트)
- **저장위치**: `output/images/briefing_card_{date}.png`

## 이미지 구성

### 헤더 섹션
- 제목: "당신이 잠든 사이"
- 날짜: YYYY.MM.DD
- 상단 라인: 초록색 강조 (#10B981)

### 좌측: 종목 정보
- 라벨: "오늘의 화제 종목"
- 종목명 (대문자)
- 종목 코드
- 주가 및 등락률
  - 상승: 초록색
  - 하락: 빨간색
- 선정 기준

### 우측: 핵심 뉴스
- 라벨: "핵심 뉴스"
- 뉴스 제목 (주요 뉴스 1개)
- 핵심 포인트 (2개)
- 시장 영향도 표시

### 하단 섹션
- 생성 시간 (자동)
- 브랜드: "While You Were Sleeping"
- 태그라인: "당신이 잠든 사이, 세상은 움직였습니다"

## 디자인 특성

### 색상 팔레트
- 배경: #0a0e27 (짙은 네이비)
- 강조: #10b981 (초록색 - 상승)
- 알림: #f59e0b (주황색 - 강조)
- 텍스트: #ffffff (흰색)
- 라벨: #888888 (회색)

### 폰트
- 제목: Arial Bold 56pt
- 헤더: Arial 40pt
- 본문: Arial 26pt
- 라벨: Arial 16pt

### 구성요소
- 그라디언트 배경
- 카드 스타일 컨테이너 (파란색 테두리)
- 색상 코딩된 배지
- 아이콘 (📊, 📰)
- 구분선 및 공백

## 관련 명령어

- `/briefing` - 마크다운 브리핑 생성
- `/briefing-report` - Word 리포트 생성
- `/briefing-data` - Excel 데이터 생성

## 자동화

배치 생성:
```bash
for ticker in TSLA NVDA AAPL MSFT GOOGL; do
  /briefing-image $ticker
done
```

## 주의사항

- 실시간 데이터는 API 통합 필요
- 현재는 하드코딩된 샘플 데이터 사용
- 고해상도 출력용 (SNS, 이메일, 광고)
- 투명 배경은 지원하지 않음 (PNG 배경)
